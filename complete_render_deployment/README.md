# TradePro - Professional Trading Bot

A comprehensive cryptocurrency trading platform with real-time signal generation, portfolio management, and professional-grade technical analysis.

## Features

- **Real-time Trading Signals**: Advanced signal generation with 90%+ confidence levels
- **Professional Dashboard**: Clean, modern interface with glassmorphism design
- **Portfolio Management**: Track performance, P&L, and risk metrics
- **Market Analysis**: Technical indicators, sentiment analysis, and volatility tracking
- **Multi-Exchange Support**: Configured for Bybit futures trading
- **Authentic Data**: CoinGecko and Coinbase API integration

## Quick Deploy to Render

### Manual Setup (Recommended)

1. **Create New Web Service**
   - Connect your GitHub repository
   - Select "Web Service"

2. **Build Configuration**
   - Build Command: `pip install -r requirements.txt`
   - Start Command: `python main.py`
   - Environment: Python 3.11

3. **Add PostgreSQL Database**
   - Create PostgreSQL service (Free tier)
   - Render automatically links DATABASE_URL

4. **Environment Variables** (Auto-configured)
   - `PORT` - Auto-generated by Render
   - `SESSION_SECRET` - Auto-generated by Render
   - `DATABASE_URL` - Auto-linked when PostgreSQL added
