Bybit API Setup (Recommended)
Step-by-Step Setup:
  1. Log into Bybit: Go to bybit.com
  2. Navigate to API: Account & Security → API Management
  3. Create New Key: Click "Create New Key"
  4. Configure Permissions:
    • ✅ Read-Only (select this)
    • ✅ Orders (query order info)
    • ✅ Positions (query positions)
    • ✅ USDC Derivatives Trading
  5. Application Name: Enter "Trading Dashboard"
  6. Copy Keys: Save both API Key and Secret Key
Benefits:
  • ✅ Real-time prices
  • ✅ Direct exchange data
  • ✅ No rate limits
  • ✅ Free to use
  • ✅ Read-only access
